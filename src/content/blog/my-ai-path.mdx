今天「珂以这样玩AI」正式上线。趁这个节点,我把这三年半的路重新走了一遍——不是凭记忆重写,而是把**当时记下的手记**串起来。你可以按时间顺序读,那是一条真实的成长轨迹:

## 这条路的地图

**[2022.12 · ChatGPT is scary good:我好像推开了一扇门](/blog/chatgpt-scary-good)**
马斯克一句「好得吓人」,让一个有二十年技术免疫力的通信老兵,一夜之间免疫失效。

**[2023.11 · ChatGPT 一周年:这一年,我把 AI 焊进了工作流](/blog/dialogue-era-one-year)**
对话时代。凡是键盘上的活,先问一句「这事能不能让 AI 干」;攒下了文字接龙、天花板和地板、提示词是方向盘这几句土话。

**[2024.11 · ChatGPT 两周年:AI 长出了手和脚](/blog/automation-to-autonomy)**
Agent 来了。从 Automation 到 Autonomy 的思维升级;在公司里把赛道收窄到只做 AI;「教」这条线悄悄发了芽。

**[2025.08 · 动手元年:我在 GitHub 放出了第一批作品](/blog/hands-on-2025)**
Claude Code 把「想法到产品」压缩到一天以内。法律 AI 打头阵,公众号「42号星际游民」开张,从 0 到 1 简单、从 1 到 100 难的教训开始上身。

**[2026.01 · 学 AI 之前,先问自己:你是哪一派?](/blog/which-camp-are-you)**
讲了几十场培训后最想放在开场的一页:悲观派、乐观派、中间派,定位决定收获。

**[2026.01 · 学 AI 这几年,我到底变了什么?](/blog/what-ai-changed-in-me)**
朋友一个问题把我问住了。答案:兴趣从缺点变资产,找到了自己的载体,从「被安排」到「能选择」。

**[2026.06 · Agent 和 Skill 这么火,为什么你还没用上一个好用的?](/blog/agent-skill-last-mile)**
最好用的 Agent 一定是定制的;通用 Agent 只解决最后一公里,而最后一公里只对走完前面九十九公里的人有价值。

## 站在今天回头看

这条路上没有顿悟时刻,没有天赋异禀。如果非要总结出一个方法,只有四个字:**当天动手。** 看到新东西,当天试;有了想法,当天做第一版;学到什么,当天用一次。三年半,复利就是这么滚出来的。

到今天,上线的应用超过 40 个,讲过的培训几十场,女娲和仓颉两个 Skill 成了我的招牌。马斯克说,AI 时代人类最需要的是保持好奇心——我用这三年半验证了这句话,外加半句:

> AI 没有门槛。你只要能想到,就可以做到。真正稀缺的,从来是想法和动手的意愿。

这条路还在继续,手记会一直写下去。跟着珂,你也可以这样玩 AI。
